Redirect Patterns

Simple Links

When you visit go/name, you are redirected to the link's target URL.

You can append a path suffix: go/name/extra/path redirects to target/extra/path.

Advanced Links

Advanced links use Go template syntax for the target URL. The following variables are available:

VariableDescription
{{"{{"}} .path {{"}}"}}Path suffix after the link name, without leading slash (e.g. foo/bar for go/name/foo/bar)
{{"{{"}} .parts {{"}}"}}Path suffix split on "/" (e.g. ["foo", "bar"])
{{"{{"}} .args {{"}}"}}Query string split on "&"
{{"{{"}} .ua {{"}}"}}User-Agent header
{{"{{"}} .email {{"}}"}}Authenticated user's email
{{"{{"}} .alias {{"}}"}}Short link name used in the request (alias name or canonical name)

Template functions

  • match "pattern" .path — partial regexp match
  • extract "pattern" .path — first submatch group
  • replace "pattern" "replacement" .path — regexp replace
