Advanced Link Syntax

Advanced links use Go template syntax for the target URL. When a user visits the link, the template is evaluated and the result is used as the redirect destination.

Template Variables

VariableTypeDescription
{{"{{"}} .path {{"}}"}} string Path suffix after the link name, without the leading slash. For go/name/foo/bar, .path is foo/bar.
{{"{{"}} .parts {{"}}"}} []string Path suffix split on /. For go/name/foo/bar, .parts is ["foo", "bar"]. Index 0 is the first segment.
{{"{{"}} .args {{"}}"}} []string Query string split on &. For go/name?a=1&b=2, .args is ["a=1", "b=2"].
{{"{{"}} .ua {{"}}"}} string The request's User-Agent header.
{{"{{"}} .email {{"}}"}} string The authenticated user's email address (empty if unauthenticated).
{{"{{"}} .alias {{"}}"}} string The short link name used in the request. When the user visits via an alias this is the alias name; when visiting directly it is the canonical link name.

Template Functions

FunctionDescriptionExample
match "pattern" string Returns true if the string matches the regexp (partial match unless anchored with ^ and $). {{"{{"}} if match "^[0-9]+$" .path {{"}}"}}...{{"{{"}} end {{"}}"}}
extract "pattern" string Returns the first submatch group. Returns an empty string if there is no match. {{"{{"}} extract "id=([0-9]+)" .path {{"}}"}}
replace "pattern" "replacement" string Replaces all matches of the regexp with the replacement string. Supports $1, $2, etc. for capture groups. {{"{{"}} replace "-" "_" .path {{"}}"}}

Examples

Simple path passthrough

https://docs.example.com/{{"{{"}} .path {{"}}"}}

go/docs/api/v2 → https://docs.example.com/api/v2

Conditional redirect based on path

{{"{{"}} if match "^[0-9]+$" .path {{"}}"}}https://issues.example.com/issue/{{"{{"}} .path {{"}}"}}{{"{{"}} else {{"}}"}}https://issues.example.com/search?q={{"{{"}} .path {{"}}"}}{{"{{"}} end {{"}}"}}

go/issue/123 → https://issues.example.com/issue/123
go/issue/login-bug → https://issues.example.com/search?q=login-bug

Extract a value from the path

https://github.com/org/repo/pull/{{"{{"}} extract "pr/([0-9]+)" .path {{"}}"}}

go/myrepo/pr/42 → https://github.com/org/repo/pull/42

Per-user redirect

https://dashboard.example.com/user/{{"{{"}} .email {{"}}"}}

Each user is sent to their own dashboard page.
